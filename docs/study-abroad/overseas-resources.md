# 侨乡资源

五邑大学坐落于中国著名侨乡——广东省江门市，拥有独特的侨乡特色和丰富的国际教育资源。作为侨乡高校，五邑大学在对外交流与合作方面具有天然优势，为学生的留学和升学提供了丰富的渠道和资源。

## 海外校友网络

### 五邑大学海外校友会

五邑大学在全球多个国家和地区设有校友会，这些校友会是留学和就业的宝贵资源。

| 地区 | 主要城市 | 联系方式 |
|------|----------|----------|
| 北美 | 纽约、旧金山、洛杉矶、多伦多、温哥华 | 通过校友会公众号联系 |
| 欧洲 | 伦敦、巴黎、柏林 | 通过校友会公众号联系 |
| 澳洲 | 悉尼、墨尔本 | 通过校友会公众号联系 |
| 东南亚 | 新加坡、吉隆坡 | 通过校友会公众号联系 |
| 港澳 | 香港、澳门 | 通过校友会公众号联系 |

### 校友资源利用建议

- **信息获取**：校友可以提供目标院校的真实就读体验和申请建议
- **推荐信**：在海外高校任教或工作的校友可以提供推荐信
- **实习机会**：校友所在公司可能提供实习或全职工作机会
- **生活帮助**：初到海外时，校友可以提供接机、租房等生活帮助
- **人脉拓展**：参加校友活动，拓展职业人脉

::: tip 如何联系校友
关注五邑大学校友会微信公众号，或通过学校就业指导中心获取校友联系方式。在联系校友时，请保持礼貌和专业，说明你的来意和需求。
:::

五邑大学与多所海外高校建立了合作关系，开展中外合作办学项目和联合培养项目。

### 项目模式说明

**2+2双学位项目**：
- 前两年在五邑大学学习，后两年赴合作院校学习
- 毕业后获得五邑大学和合作院校的双学士学位
- 适合大一入学时申请

**3+1单学位项目**：
- 前三年在五邑大学学习，最后一年赴合作院校学习
- 毕业后获得五邑大学学士学位和合作院校的硕士学位（部分项目）
- 适合大三时申请

**交换生项目**：
- 交换期通常为1个学期或1学年
- 免交对方学校学费，只需缴纳五邑大学学费
- 学分互认

### 申请条件

- GPA要求：通常要求3.0+/4.0（具体以各项目要求为准）
- 语言要求：IELTS 6.0-6.5 或 TOEFL 75-90
- 无违纪记录
- 通过学校组织的选拔面试

## 粤港澳联合培养项目

五邑大学积极参与粤港澳大湾区的教育合作，与11所港澳高校建立了联合培养关系。

### 参与高校

香港地区：香港大学、香港中文大学、香港科技大学、香港城市大学、香港理工大学、香港浸会大学、香港岭南大学、香港教育大学

澳门地区：澳门大学、澳门科技大学、澳门城市大学

### 项目优势

1. **学分互认**：在港澳高校修读的学分可转换为五邑大学学分
2. **费用减免**：部分项目享受学费减免或奖学金
3. **双导师制**：由五邑大学和港澳高校的教授共同指导
4. **资源共享**：可使用港澳高校的图书馆、实验室等资源
5. **升学便利**：联合培养经历有助于后续申请港澳高校的研究生

::: info 报名方式
关注五邑大学国际交流处官网和微信公众号，每年春秋两季发布报名通知。建议提前准备语言成绩。
:::

### 侨乡基金会资源

江门五邑地区有众多华侨华人基金会，部分基金会提供教育资助：

- **江门市侨联**：可提供海外侨胞资源对接
- **五邑大学教育发展基金会**：管理各类校友和社会捐赠奖学金
- **各区县侨务部门**：可协助联系海外乡亲资源

## 如何充分利用侨乡资源

### 在校期间

1. **关注国际交流处**：及时获取合作项目和交换生信息
2. **参加侨乡文化活动**：拓展国际视野，了解华侨历史
3. **联系海外校友**：通过校友会建立海外联系
4. **学好英语**：语言能力是利用国际资源的基础
5. **保持高GPA**：大部分项目和奖学金都有GPA要求

### 申请期间

1. **利用校友推荐**：请海外校友帮忙了解目标院校信息或提供推荐信
2. **申请联合培养**：通过联合培养项目提升背景
3. **申请奖学金**：充分利用侨乡特色奖学金减轻经济负担
4. **参加宣讲会**：合作院校经常来校举办宣讲会，面对面交流

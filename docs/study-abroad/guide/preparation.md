# 申请准备

留学申请是一场持久战，越早规划，胜算越大。本指南将为你梳理从大一到大四的完整申请时间线，帮助你合理分配精力，稳步推进申请进程。

## 整体时间规划

<Timeline>
  <TimelineItem title="大一：打好基础" time="大一上~大一下">
    <ul>
      <li>保持高GPA，争取每学期绩点在专业前20%</li>
      <li>打好英语基础，尽早通过四六级考试</li>
      <li>了解留学基本概念，关注学长学姐的申请经验</li>
      <li>参加学术讲座和社团活动，拓展视野</li>
    </ul>
  </TimelineItem>
  <TimelineItem title="大二：背景提升" time="大二上~大二下">
    <ul>
      <li>继续保持GPA，专业课争取高分</li>
      <li>开始准备标化考试（TOEFL/IELTS），了解考试形式</li>
      <li>主动联系导师，争取加入科研项目</li>
      <li>参加学科竞赛（数学建模、编程竞赛等）</li>
      <li>利用寒暑假参加实习或短期交流项目</li>
    </ul>
  </TimelineItem>
  <TimelineItem title="大三：全力冲刺" time="大三上~大三下">
    <ul>
      <li>确定申请方向（国家、专业、学位类型）</li>
      <li>完成标化考试，争取达到目标分数</li>
      <li>深入参与科研项目，争取发表论文或申请专利</li>
      <li>利用暑假参加高含金量实习</li>
      <li>开始研究目标院校，建立选校名单</li>
      <li>大三下学期开始准备推荐人，与教授建立良好关系</li>
    </ul>
  </TimelineItem>
  <TimelineItem title="大四：申请季" time="大四上~大四下">
    <ul>
      <li>9-10月：确定最终选校名单，开始撰写文书</li>
      <li>10-12月：提交申请（注意各校截止日期）</li>
      <li>次年1-3月：准备面试，等待录取结果</li>
      <li>次年3-4月：比较offer，做出最终选择</li>
      <li>次年4-5月：办理签证，安排住宿</li>
      <li>次年6-8月：行前准备，飞赴海外</li>
    </ul>
  </TimelineItem>
</Timeline>

## 背景提升优先级

在留学申请中，各项背景的权重因国家和专业而异，但总体优先级如下：

### 第一梯队：GPA

GPA 是所有申请的基石。无论是申请硕士还是博士，GPA 都是最重要的硬性指标之一。

- **目标建议**：申请美国Top 30建议GPA 3.5+/4.0，Top 50建议3.3+/4.0
- **提升策略**：
  - 重视专业课成绩，尤其是与申请方向相关的课程
  - 合理选课，避免因难度过高导致低分
  - 利用重修机会提高不理想的课程成绩（注意各校政策）
  - 如果学校提供WES认证，可以尝试通过认证提高GPA

### 第二梯队：标化考试

标化考试是申请的门槛性指标，达到基本要求后，边际效益递减。

- **语言考试**：TOEFL 100+ / IELTS 7.0+ 为大多数项目的安全线
- **研究生入学考试**：GRE 320+ / GMAT 700+ 为有竞争力的分数
- **备考建议**：
  - 提前3-6个月开始备考
  - 先攻克语言考试，再准备GRE/GMAT
  - 利用暑假集中备考效率最高

### 第三梯队：科研经历

对于申请研究型硕士和博士项目，科研经历至关重要。

- **本科生科研途径**：
  - 联系本院系教授，加入课题组
  - 参加大学生创新创业训练计划（大创）
  - 申请暑期科研（REU、海外暑研）
  - 发表论文或会议摘要
- **科研质量 > 数量**：一段深入的科研经历比多段浅层参与更有价值

### 第四梯队：实习经历

对于申请授课型硕士和就业导向项目，实习经历非常重要。

- **实习建议**：
  - 优先选择与申请方向相关的实习
  - 大公司实习 > 小公司实习（但也要看具体工作内容）
  - 至少2段实习经历，每段不少于2个月
  - 争取获得推荐信

### 第五梯队：课外活动

课外活动对申请的直接影响较小，但可以体现综合素质。

- **有价值的活动**：学生会/社团干部、志愿者活动、国际交流经历
- **注意**：活动贵精不贵多，深度参与比广泛涉猎更有说服力

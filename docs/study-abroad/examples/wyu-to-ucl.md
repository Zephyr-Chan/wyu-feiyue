## 选校策略

### 选校原则

我的选校原则是"冲刺-匹配-保底"三层结构：

1. **冲刺校（2所）**：UCL、爱丁堡大学
2. **匹配校（3所）**：曼彻斯特大学、伦敦国王学院（KCL）、布里斯托大学
3. **保底校（2所）**：谢菲尔德大学、格拉斯哥大学

### 为什么选择英国？

选择英国读研主要有以下几个原因：

- **学制短**：英国授课型硕士仅需1年，时间成本较低
- **G5名校**：UCL是世界顶尖大学，QS排名全球第9
- **PSW签证**：毕业后可获得2年工作签证
- **费用可控**：相比美国，英国的整体费用相对较低
- **文化吸引**：对英国文化和历史有浓厚兴趣

### 为什么选择UCL？

选择UCL的原因：

- **排名高**：QS 2024全球排名第9，计算机学科排名前列
- **地理位置**：位于伦敦市中心，就业机会丰富
- **课程设置**：UCL的MSc Computer Science课程灵活，可以选择AI、安全、软件工程等多个方向
- **研究实力**：UCL在计算机视觉、机器学习等领域有很强的研究实力

## 文书写作经验

### Personal Statement

PS是英国申请中最重要的文书。我的PS结构如下：

**第一段：引言**
用一个具体的故事引入，讲述我是如何对计算机科学产生兴趣的。我选择了大二时参与图像分类项目的经历作为切入点。

**第二段：学术背景**
介绍我的核心课程成绩和学术能力，强调与申请方向相关的课程（数据结构、算法、机器学习等）。

**第三段：科研经历（核心段落）**
详细描述了我在智能计算实验室的经历，包括：
- 研究问题：基于深度学习的医学图像分类
- 我的贡献：改进了ResNet模型，在数据集上实现了3%的精度提升
- 成果：发表了EI会议论文

**第四段：实习经历**
描述了在互联网公司的后端开发实习，展示了我的工程实践能力。

**第五段：为什么选择UCL**
具体说明了UCL的哪些方面吸引了我：
- 提到了UCL计算机科学系的某个具体研究方向
- 引用了某位教授的论文
- 说明了伦敦的地理位置优势

**第六段：职业规划**
描述了我的短期目标（毕业后在英国或回国从事AI研发工作）和长期目标（成为AI领域的技术专家）。

### PS写作建议

1. **早开始**：至少提前2个月开始写PS
2. **多修改**：我的PS修改了十多稿，每稿都有明显进步
3. **找人审阅**：请英语好的同学、老师或专业机构帮忙审阅
4. **定制化**：虽然英国申请可以一篇PS投多所学校，但我还是为UCL单独定制了一版
5. **具体化**：用具体事例代替空泛的描述

### 推荐信

我的两封推荐信分别来自：

1. **科研导师**：重点描述了我的科研能力和学术潜力
2. **专业课教授**：重点描述了我的课堂表现和学习能力

建议提前至少2个月联系推荐人，为他们准备详细的素材（CV、成绩单、与推荐人互动的具体事例等）。

## 面试经验

UCL的MSc Computer Science项目有面试环节。我的面试是通过Microsoft Teams进行的，时长约20分钟。

### 面试内容

1. **自我介绍**（2-3分钟）
2. **学术问题**：
   - 请解释什么是过拟合，如何解决？
   - 请描述一下你论文中使用的模型架构。
   - 你对哪个计算机科学领域最感兴趣？为什么？
3. **行为问题**：
   - 你为什么选择UCL？
   - 你在团队合作中遇到过什么挑战？
4. **自由提问**（2-3分钟）

### 面试准备建议

- **复习专业知识**：重点复习数据结构、算法、机器学习等核心课程
- **了解UCL**：深入研究UCL计算机科学系的课程设置和教授研究方向
- **模拟面试**：找同学进行模拟面试，练习表达
- **准备问题**：准备2-3个可以问面试官的问题

## 经验总结

### 成功经验

1. **GPA是王道**：保持高GPA是最重要的，这是我申请成功的基础
2. **科研经历加分**：虽然论文是第三作者，但展示了我有真实的科研经历
3. **早规划早准备**：从大二就开始准备，给了我充足的时间提升背景
4. **IELTS尽早达标**：提前考出语言成绩，避免申请季手忙脚乱
5. **PS用心打磨**：PS是展示个人特色的最好机会，一定要认真对待

### 失败教训

1. **爱丁堡Waitlist**：可能是因为PS中对爱丁堡的针对性不够强
2. **IELTS写作偏低**：写作只有6.5分，虽然达到了UCL的要求（6.5），但部分学校要求7.0
3. **实习时间偏短**：只有一段3个月的实习，如果能有两段会更好

### 给学弟学妹的建议

1. **不要因为双非而自我设限**：五邑大学的背景确实不是优势，但绝不是障碍。关键在于你的个人实力和申请策略。
2. **GPA > 一切**：在GPA和科研/实习之间，优先保证GPA。
3. **尽早确定方向**：越早确定留学方向，准备时间越充分。
4. **善用侨乡资源**：五邑大学的侨乡特色和中外合作项目是独特的优势，充分利用这些资源。
5. **保持心态平和**：申请过程中会有焦虑和不确定性，保持积极的心态很重要。
6. **多交流多请教**：多和有经验的学长学姐交流，他们的建议非常有价值。

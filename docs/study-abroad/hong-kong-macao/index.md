# 港澳升学经验

港澳高校因其国际化程度高、地理位置近、学费相对合理等优势，成为越来越多内地学生的升学选择。五邑大学地处粤港澳大湾区，与港澳高校交流密切，拥有独特的地缘优势。本板块汇集了五邑大学学长学姐们在港澳升学的经验分享。

## 港澳升学概况

### 为什么选择港澳？

- **地理优势**：毗邻广东，交通便捷，文化相近
- **名校云集**：香港5所大学进入QS世界排名前100（港大、港中文、港科大、港城大、港理大）
- **国际化教育**：全英文授课，国际师资，全球视野
- **费用适中**：相比英美，港澳的学费和生活费更为合理
- **就业前景**：粤港澳大湾区就业机会丰富，IANG签证方便留港工作
- **升学跳板**：港澳学位受国际认可，便于继续申请海外博士

### 粤港澳联合培养项目

五邑大学与多所港澳高校建立了联合培养项目，这是五邑大学学生的独特优势。

#### 项目介绍

粤港澳联合培养项目是广东省教育厅推动的高等教育合作项目，允许学生在五邑大学和港澳高校之间进行学分互认和联合培养。

### 香港升学 vs 澳门升学

| 对比维度 | 香港 | 澳门 |
|----------|------|------|
| 世界排名 | 整体更高 | 相对较低 |
| 学费（年均） | HKD 140,000-180,000 | MOP 100,000-130,000 |
| 生活费（年均） | HKD 60,000-100,000 | MOP 40,000-60,000 |
| 就业机会 | 更多（金融、科技） | 相对较少 |
| IANG签证 | 有 | 无（但可申请其他工作签证） |
| 申请难度 | 较高 | 相对较低 |

## 港澳升学常见问题

### Q: 港澳高校对五邑大学的认可度如何？

五邑大学在粤港澳地区有一定的知名度，尤其是通过联合培养项目与港澳高校建立了良好的合作关系。申请时，个人实力（GPA、语言成绩、科研经历）比学校背景更重要。

### Q: 港澳读研后可以留在当地工作吗？

- **香港**：非本地毕业生可通过IANG签证在香港停留12个月（2022年起延长至2年），期间可自由找工作。
- **澳门**：毕业后可通过"外地雇员身份认别证"在澳门工作，但需要雇主担保。

### Q: 港澳学位在国内考公有优势吗？

港澳高校的学位受教育部认可，可以在中国留学生服务中心进行学历认证。考公、考编、落户等均认可港澳学历。

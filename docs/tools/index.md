# 实用工具

留学申请过程中需要使用大量在线工具和网站。本页面整理了各类实用工具的链接和简介，帮助你高效完成申请的各个环节。

## 世界大学排名

选择目标院校时，参考多个排名体系可以获得更全面的视角。

| 排名 | 说明 | 链接 |
|------|------|------|
| QS世界大学排名 | 最受关注的综合排名，注重学术声誉和雇主声誉 | [topuniversities.com](https://www.topuniversities.com/world-university-rankings) |
| THE世界大学排名 | 注重教学、研究、引文、国际化和产业收入 | [timeshighereducation.com](https://www.timeshighereducation.com/world-university-rankings) |
| ARWU（软科） | 注重科研成果和学术表现（诺贝尔奖、论文发表等） | [shanghairanking.com](https://www.shanghairanking.com) |
| US News全球排名 | 注重学术研究表现和全球及地区声誉 | [usnews.com/education](https://www.usnews.com/education/best-global-universities) |
| CSRankings | 计算机科学领域专业排名，按论文发表量排名 | [csrankings.org](https://csrankings.org) |

::: tip 排名参考建议
不要只看一个排名，建议综合参考多个排名。不同排名的侧重点不同，选择与自己关注点匹配的排名。专业排名有时比综合排名更有参考价值。
:::

## 推荐工具组合

### 申请季必备

1. **选校**：QS排名 + Niche + 一亩三分地
2. **备考**：ETS官网 + Anki + Grammarly
3. **文书**：Overleaf + Google Docs + Hemingway Editor
4. **申请**：各校网申系统 + 推荐信管理表格
5. **签证**：DS-160 + USTravelDocs + SEVIS
6. **租房**：Student.com / 异乡好居

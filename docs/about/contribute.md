# 投稿指南

五邑大学飞跃手册是一个由学生自发维护的开源项目，旨在帮助更多五邑大学同学实现升学和就业的梦想。我们欢迎所有五邑大学的在校生和校友投稿分享自己的经验。

## 投稿流程

以下是投稿的完整工作流程：

<Timeline>
  <TimelineItem title="Fork 项目" time="第1步">
    <p>访问五邑大学飞跃手册的 GitHub 仓库，点击右上角的 Fork 按钮，将项目复制到你自己的 GitHub 账号下。</p>
  </TimelineItem>
  <TimelineItem title="创建分支" time="第2步">
    <p>将 Fork 后的仓库克隆到本地，创建一个新的分支（如 <code>add-my-experience</code>），在分支上进行修改，保持主分支干净。</p>
  </TimelineItem>
  <TimelineItem title="撰写文章" time="第3步">
    <p>参考投稿模板，在对应的目录下创建 Markdown 文件，撰写你的经验分享文章。确保内容真实、原创、有参考价值。</p>
  </TimelineItem>
  <TimelineItem title="提交 PR" time="第4步">
    <p>将修改推送到你的 GitHub 仓库，然后向主仓库提交 Pull Request，详细描述你的更改内容。</p>
  </TimelineItem>
  <TimelineItem title="审核修改" time="第5步">
    <p>维护者会在 3-5 个工作日内审核你的 PR，如有需要修改的地方会在 PR 中留言反馈。</p>
  </TimelineItem>
  <TimelineItem title="发布上线" time="第6步">
    <p>审核通过后，PR 会被合并到主分支，文章将在下次站点构建后自动发布上线。</p>
  </TimelineItem>
</Timeline>

## 投稿方式

我们提供四种投稿方式，你可以选择最方便的一种。

### 方式一：GitHub Pull Request（推荐）

如果你熟悉Git和GitHub，推荐使用PR方式投稿，这样可以保留完整的修改记录。

**步骤**：

1. **Fork项目**：访问五邑大学飞跃手册的GitHub仓库，点击右上角的Fork按钮
2. **Clone到本地**：将Fork后的仓库克隆到本地
   ```bash
   git clone https://github.com/你的用户名/wyu-feiyue.git
   cd wyu-feiyue
   ```
3. **创建新分支**：
   ```bash
   git checkout -b add-my-experience
   ```
4. **创建文章**：在对应的目录下创建Markdown文件（如 `docs/study-abroad/north-america/my-story.md`）
5. **提交更改**：
   ```bash
   git add .
   git commit -m "添加北美留学经验分享"
   git push origin add-my-experience
   ```
6. **提交PR**：在GitHub上创建Pull Request，描述你的更改

### 方式二：在线表单

如果你不熟悉Git，可以使用在线表单投稿。

**步骤**：

1. 访问[投稿表单](#)（链接待补充）
2. 填写你的基本信息和文章内容
3. 上传文章文件（Markdown格式）
4. 提交表单

我们会在收到表单后的3个工作日内处理你的投稿。

## 审核流程

1. **收到投稿**：我们会在3个工作日内确认收到你的投稿
2. **内容审核**：审核内容包括：
   - 内容的真实性和准确性
   - 文章结构和逻辑
   - 语言表达和格式规范
   - 是否包含不当内容
3. **修改反馈**：如需修改，我们会在5个工作日内给出修改建议
4. **发布上线**：审核通过后，文章将在7个工作日内发布到网站上
5. **通知作者**：文章发布后，我们会通过邮件或微信通知你

## 投稿要求

### 内容要求

- **真实性**：文章内容必须真实，不得虚构或夸大
- **原创性**：文章必须为原创，不得抄袭他人作品
- **实用性**：文章应对读者有实际参考价值
- **正面导向**：文章应积极向上，传递正能量

### 格式要求

- **字数**：建议2000-5000字
- **格式**：Markdown格式
- **图片**：如有图片，请使用相对路径引用，图片放在 `docs/public/images/` 目录下
- **隐私**：不要泄露他人的个人隐私信息

### 不接受的内容

- 包含虚假信息的内容
- 涉及政治敏感话题的内容
- 包含广告或商业推广的内容
- 抄袭或洗稿的内容
- 攻击或贬低他人的内容

## 常见问题

### Q: 我不是五邑大学的学生可以投稿吗？

目前我们主要接受五邑大学在校生和校友的投稿。如果你是其他学校的同学，欢迎参考我们的投稿指南创建自己学校的飞跃手册。

### Q: 文章可以匿名发布吗？

可以。如果你希望匿名发布，请在投稿时说明，我们会使用你提供的昵称。

### Q: 投稿有稿费吗？

目前飞跃手册是一个纯公益项目，暂无稿费。但你的经验分享将帮助到许多学弟学妹，这本身就是最大的价值。

### Q: 多久能发布？

从投稿到发布通常需要1-3周，具体取决于审核进度和修改次数。

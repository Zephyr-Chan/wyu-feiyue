## 关于本项目

**邑大飞跃手册**是由五邑大学学生自发组织编写的升学/留学经验分享平台，灵感来源于清华大学、南方科技大学、华南理工大学等高校的飞跃手册项目。

我们的使命是**打破信息壁垒**，帮助邑大学子在升学和留学申请中获得更好的信息支持。

### 项目特色

- **根植侨乡**：充分利用五邑大学侨乡背景，汇聚海外校友资源
- **多方向覆盖**：留学、保研、考研、就业、考公五大方向全覆盖
- **开放协作**：基于GitHub开源协作，支持多种投稿方式
- **持续更新**：编委会制度保障，每年毕业季集中更新

### 参与贡献

我们欢迎所有邑大校友和在校生参与贡献！你可以通过以下方式参与：

- **分享经验**：提交你的升学/留学/就业经验分享
- **反馈建议**：在 GitHub 上提交 Issue 提出改进建议
- **推广传播**：将手册推荐给有需要的同学
- **技术贡献**：参与网站开发和维护

> **提示**：所有投稿均支持实名或匿名发布。查看 [投稿指南](/about/contribute) 了解详情。

# 保研经验

保研（推荐优秀应届本科毕业生免试攻读研究生）是国内读研的重要途径之一。通过保研可以直接进入心仪的高校攻读硕士或博士研究生，无需参加全国硕士研究生统一招生考试。本板块汇集了五邑大学学长学姐们的保研经验分享。

## 保研政策介绍

### 什么是保研？

保研全称为"推荐优秀应届本科毕业生免试攻读研究生"，是指经教育部批准的高校，可以推荐本校优秀应届本科毕业生免试攻读硕士研究生（包括直博生）。

### 保研基本条件

- **学业要求**：本科前三年（或前五学期）平均学分绩点排名在本专业前列
- **英语要求**：通常要求通过CET-4/CET-6（具体分数要求因学校而异）
- **综合素质**：无违纪记录，积极参与科研、竞赛、社会实践等活动
- **推免资格**：获得所在学校的推免名额

### 五邑大学保研情况

五邑大学每年有一定数量的推免名额，具体名额由教育部下达。一般要求：

- GPA排名在本专业前5%-10%（具体比例以当年通知为准）
- 通过CET-4（部分学院要求CET-6）
- 无补考和重修记录（或重修后成绩达到要求）

::: tip 关键信息
推免名额和具体要求每年可能有所调整，请以学校教务处当年发布的正式通知为准。
:::

## 夏令营攻略

### 什么是夏令营？

夏令营是高校在暑假期间举办的招生选拔活动，通常为期3-7天。通过夏令营可以提前了解目标院校，与导师面对面交流，表现优异者可获得"优秀营员"资格。

### 夏令营申请材料

- **个人简历**：突出学业成绩、科研经历、竞赛获奖
- **个人陈述**：阐述学术兴趣、研究计划和申请动机
- **成绩单**：前五学期官方成绩单
- **排名证明**：由学院出具的官方排名证明
- **推荐信**：2封教授推荐信
- **英语成绩**：CET-4/CET-6成绩单
- **获奖证书**：竞赛获奖、奖学金等证明材料
- **科研成果**：论文、专利、项目报告等

### 夏令营面试准备

1. **自我介绍**：准备中英文版本的3分钟自我介绍
2. **专业知识**：复习核心专业课，尤其是与申请方向相关的课程
3. **科研经历**：能够详细介绍参与过的科研项目
4. **英语能力**：准备英文问答环节
5. **了解导师**：提前了解目标导师的研究方向和近期论文

### 夏令营申请策略

- **广泛申请**：同时申请多所学校的夏令营，增加机会
- **分层定位**：冲刺校、匹配校、保底校各申请2-3所
- **关注时间**：不同学校的申请截止日期不同，注意及时提交
- **提前联系导师**：部分学校允许提前联系导师，表达兴趣

## 预推免攻略

### 什么是预推免？

预推免是高校在正式推免系统开放前进行的招生选拔活动，通常在8-9月进行。预推免是夏令营的补充，适合没有参加夏令营或夏令营未获得满意结果的同学。

### 预推免 vs 夏令营

| 对比维度 | 夏令营 | 预推免 |
|----------|--------|--------|
| 时间 | 5-7月 | 8-9月 |
| 形式 | 线下为主，3-7天 | 线上/线下，1-2天 |
| 内容 | 学术活动+面试 | 以面试为主 |
| 名额 | 较多 | 相对较少 |
| 竞争 | 相对较小 | 较大 |

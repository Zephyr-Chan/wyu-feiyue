## 数据说明

::: tip 数据来源
以上统计数据来源于邑大校友投稿和公开信息整理，仅供参考。实际录取情况因个人背景、申请年份等因素而异。
:::

## 贡献数据

如果你有录取数据愿意分享，欢迎通过以下方式贡献：

1. **投稿分享**：在经验分享文章中包含你的录取结果
2. **数据反馈**：发送邮件至 wyu-feiyue@outlook.com
3. **GitHub Issue**：在仓库中提交数据更正建议

你的每一次贡献都将帮助更多邑大学子做出更好的升学决策。
